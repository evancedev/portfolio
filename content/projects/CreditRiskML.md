Developed an end-to-end machine learning pipeline that predicts loan default probability for Kenya's fintech ecosystem. The system addresses the challenge of credit risk assessment in markets with limited traditional credit histories.

Key features:

- 91.67% accuracy with ROC-AUC score of 0.9856 using XGBoost
- Comprehensive EDA with statistical analysis and visualization
- Advanced handling of class imbalance using SMOTE
- Hyperparameter optimization with RandomizedSearchCV
- Interactive Streamlit web app with KES currency support
- Batch processing for portfolio analysis

This solution enables financial institutions to make same-day, data-driven loan decisions, significantly reducing manual processing time and improving portfolio management.
